# V1 Pilot Run Archive

This directory contains results from the initial 40-question pilot run completed on 2025-11-30.

## Summary
- **40 questions** across 4 categories
- **20% overall hallucination rate**
- **70% hallucination rate** on nonexistent entities
- **Oppositeness metric** showed promising trend (r=0.285, p=0.075)

## Files
- `processed/` - Intermediate data (embeddings, judgments, geometry features)
- `results/` - Final merged dataset and visualizations
- `ANALYSIS_RESULTS.md` - Complete analysis of findings

This served as proof-of-concept for the V2 expanded experiment (480 questions, multi-model, predictive modeling).
